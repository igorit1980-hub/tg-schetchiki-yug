# Channel Setup Checklist

## Telegram channel creation

- Create a new Telegram channel
- Set channel name
- Set public channel mode
- Reserve username
- Add avatar
- Add channel description
- Reduce reactions if needed
- Disable comments if needed

## Product filling

- Select first 5 water meter products
- Select first 3 to 5 gas meter products
- Select first 3 to 5 pump products
- Select first 3 to 5 accessory products
- Prepare photo for each product
- Prepare short card text for each product
- Publish first 15 to 20 product posts

## Product post requirements

- Short title
- Visible price
- 3 to 5 advantages
- Stock or order status
- Delivery note
- Call to action
- At least 1 clean product photo

## Promotions and service content

- Publish promotions post
- Publish bestsellers post
- Publish wholesale order post
- Publish manager contact post
- Publish company post

## Launch readiness check

- Name is correct
- Description is added
- Avatar is uploaded
- Username works
- Main pinned post exists
- Main pinned post is pinned
- Category posts are published
- Internal links work
- Promotions post exists
- Bestsellers post exists
- Wholesale post exists
- Manager contact post exists
- About company post exists
- First 15 to 20 product posts are published
- Channel looks like a storefront, not like a news feed

## Manual QA before showing to others

- Open the channel on phone
- Open the channel on desktop
- Check readability of pinned post
- Check readability of product cards
- Check all post links
- Check ordering flow clarity
- Check whether manager contact is visible enough
- Check whether prices are visible enough

## Typical launch fixes

- Remove weak or empty categories
- Add missing prices
- Add missing photos
- Shorten overlong texts
- Strengthen calls to action
- Clarify order route

## Immediate actions after channel creation

- Send channel link to yourself
- Send channel link to team
- Test scrolling and navigation
- Collect first comments
- Fix weak spots immediately

- Improve catalog UI
- Improve customer card UI
- Add live/fallback badge in Mini App
- Create Telegram bot
- Connect channel to bot
