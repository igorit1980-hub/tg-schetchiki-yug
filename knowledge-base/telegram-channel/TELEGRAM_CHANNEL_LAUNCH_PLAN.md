# Telegram Channel Launch Plan

## Goal

Create a Telegram channel for `Счетчики Юг` in the style of a clean product storefront similar to BigSaleApple, but adapted for engineering equipment and B2B/B2C sales flows.

The channel is needed for:
- showing products
- showing promotions
- showing curated selections
- routing people to manager contact
- later routing people to the Telegram bot and Mini App

Important architecture:
- channel = storefront and traffic source
- bot = entry point
- Mini App = catalog and actions
- Bitrix24 = CRM, requests, statuses

## Channel concept

Correct channel format:
- product catalog channel
- not a news channel

Core content types:
- catalog sections
- promotions
- popular products
- manager contact
- wholesale offers
- later links to Mini App

Target audience:
- монтажники
- проектировщики
- застройщики
- управляющие компании
- оптовые покупатели

Tone of voice:
- businesslike
- concise
- sales-oriented
- no entertainment style

## Launch stages

### Stage 1. Prepare the concept

1. Fix the main goal of the channel:
   product display and routing into manager request and later Mini App.
2. Confirm the format:
   product-catalog storefront channel.
3. Fix the target audience:
   installers, designers, developers, УК, wholesale clients.

### Stage 2. Create the Telegram channel

1. Open Telegram
2. Create a new channel
3. Set the channel name
4. Add description
5. Upload avatar
6. Choose public channel mode
7. Set username

Recommended initial settings:
- public channel
- minimal reactions
- comments disabled or minimized at launch

## Style rules

The channel should feel like a compact storefront.

Each post should usually contain:
- emoji in the first line
- short title
- price
- 3 to 5 selling points
- stock information
- call to action

Avoid:
- long walls of text
- clutter
- news-style or entertainment-style posts

Reference principle:
- short
- fast
- clear
- easy to scroll

## Minimum launch checklist

Before showing the channel to anyone, verify that it already has:
- channel name
- avatar
- description
- pinned main catalog post
- 5 to 6 category posts
- 15 to 20 product posts
- promotion post
- wholesale post
- manager contact post

If all of this exists, the channel is ready for first use.

## Immediate QA after launch

Test the channel manually:
- is navigation clear
- do category links work
- does it feel like a storefront
- are prices visible
- is it obvious how to order

Typical weak points to fix immediately:
- unnecessary categories
- not enough photos
- unclear prices
- unclear ordering flow

## Minimal 10-step route

1. Create the channel

5. Create the pinned main catalog post
6. Create 5 to 6 category posts
7. Copy category links
8. Insert category links into the pinned post

## Project connection to Mini App

This Telegram channel does not replace the Mini App.

Correct future flow:
- channel works as a storefront and traffic layer
- channel leads into bot or Mini App
- Mini App handles catalog actions
- backend sends requests into Bitrix24
