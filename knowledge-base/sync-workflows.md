# Sync Workflows

## 1. Общий принцип

Нужен один слой интеграции, который:

- читает `Битрикс24`
- читает каталог сайта
- нормализует данные через `xml_id`
- публикует Telegram storefront JSON

## 2. Направление Site -> Knowledge Base

Сайт отдает:

- товарную карточку
- цены
- остатки
- URL
- изображение
- категорию

Правило:

- если `active=false`, товар в витрину не попадает

## 3. Направление Bitrix24 -> Knowledge Base

`Битрикс24` отдает:

- популярные товары
- акции
- период показа
- тексты для витрины
- ответственного менеджера

Правило:

- запись проходит только если она активна по стадии, категории и дате

## 4. Направление Knowledge Base -> Telegram

В Telegram уходит только итоговый слой:

- `popular_products`
- `promotions`
- `customer_status`
- `client_card`

Telegram не должен напрямую читать `Битрикс24` и не должен напрямую работать с каталогом сайта.

## 4.1. Направление Telegram -> Bitrix24

По клиентским заявкам Telegram должен:

- создать или обновить клиента в `Битрикс24`
- передать телефон, Telegram-идентификаторы и тип клиента
- получить обратно канонический статус карты и разрешенного типа цены

Правило:

- решение об оптовом доступе принимает только `Битрикс24`

## 5. Полная пересборка

Каждая синхронизация выполняется полностью:

1. загрузить все страницы `popular_products`
2. загрузить все страницы `promotions`
3. отфильтровать по stage/category/date/xmlId
4. удалить дубли по `xmlId`
5. сделать join с сайтом
6. удалить невалидные записи
7. собрать новый JSON
8. опубликовать новый JSON атомарно

## 6. Обратное обращение к данным

База знаний должна позволять:

- по `xml_id` найти товар сайта
- по `xml_id` найти управляющую запись в `Битрикс24`
- по `xml_id` понять, попадает ли товар сейчас в Telegram
- по записи `Битрикс24` найти итоговую витринную карточку

## 7. Логи ошибок

Обязательные коды:

- `EMPTY_XML_ID`
- `EMPTY_REQUIRED_BITRIX_FIELD`
- `OUT_OF_DATE_RANGE`
- `INVALID_DATE_RANGE`
- `NOT_FOUND_ON_SITE_BY_XML_ID`
- `DUPLICATE_SITE_XML_ID`
- `DUPLICATE_BITRIX_XML_ID`
- `SITE_ITEM_INACTIVE`
- `MISSING_REQUIRED_SITE_FIELDS`
- `MISSING_PROMO_PRICE`
- `MISSING_OLD_PRICE_FOR_PROMO`
- `INVALID_PROMO_PRICE`
- `BITRIX_API_ERROR`
- `SITE_SOURCE_ERROR`
- `JSON_PUBLISH_ERROR`
